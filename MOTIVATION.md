# Motivation

ViperPad is meant to grow into a fast, focused developer pad rather than a heavyweight all-in-one IDE.

## Project goals

- Excellent editing first: fast startup, smooth large-file editing, strong syntax highlighting, search, split views, and tab/workspace support.
- An instant preview loop for Markdown, HTML, text, and other developer-friendly formats with live refresh.
- Local-first developer utility: open files quickly, edit safely, watch for changes, inspect outputs, and share viewer links when useful.
- Sharp code tools without bloat: formatting, diagnostics, symbol navigation, git diff views, and snippets.
- A practical "scratchpad for real projects" feel somewhere between a notepad, previewer, and lightweight IDE.
